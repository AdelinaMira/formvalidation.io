Skip validation on specific submit button

← Examples

{% include share.html %}

Your form might have multiple submit buttons which have type="submit" attribute. By default, clicking on any submit button will perform validation and then submit the form if all the fields are valid.

If you want to skip the validation on specific submit button, just use the button option:

{% highlight javascript %} button: { // The CSS selector indicates the button selector: '[type="submit"]', // The CSS class which is added to the button when it is disabled disabled: '' } {% endhighlight %}

The following employment form has two submit buttons, but the validation is only triggered when clicking on the Validate and Submit button:
