Basic usage

The following example demonstrates how to use FormValidation to validate panels structured by Fuel UX Wizard.

We need to trigger the actionclicked.fu.wizard and finished.fu.wizard events to validate the selected panel.

You should look at the wizard section when integrating FormValidation with wizard-like components

If you want to use more Fuel UX controls, look at this example.

Not validating current step when going to previous one

If you don't want the current step to be validated when clicking the Prev button to jump to the previous step, you can add a simple check in actionclicked.fu.wizard event handler:

{% highlight javascript %} $('#orderWizard') .wizard() // Triggered when clicking the Next/Prev buttons .on('actionclicked.fu.wizard', function(e, data) { if (data.direction === 'previous') { // Do nothing if you're going to the previous step return; } ... }) .on('finished.fu.wizard', function(e) { ... }); {% endhighlight %}
