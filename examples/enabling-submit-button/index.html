Always enable the submit button

By default, the submit buttons will be disabled if there is at least one invalid field.

If you want to enable the submit buttons all the time, triggering the err.field.fv, success.field.fv event and using the disableSubmitButtons() method:

{% include demo/tabs.html id="always" frameworks="bootstrap" dir="always" codes="programmatic" %}

If you want the submit button to be enabled only after clicking it, use the getSubmitButton() method to check whether or not the submit button is clicked:

{% highlight javascript %} $(document).ready(function() { $('#alwaysEnableButtonForm') .formValidation({ ... }) .on('err.field.fv', function(e, data) { if (data.fv.getSubmitButton()) { data.fv.disableSubmitButtons(false); } }) .on('success.field.fv', function(e, data) { if (data.fv.getSubmitButton()) { data.fv.disableSubmitButtons(false); } }); }); {% endhighlight %}

Only enable the submit button if all fields are valid

We can use the isValid() method to check if all fields are valid.

The isValid() method returns correct value if the form is already validated (by either pressing the submit button or calling validate() directly). This example is supposed to work if all fields are empty which is mostly suitable for creating record form.
