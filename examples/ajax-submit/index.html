Using Ajax to submit the form

← Examples

{% include share.html %}

By triggering the success.form.fv event, we can use Ajax to submit the form data.

{% highlight javascript %} $(document).ready(function() { $(form) .formValidation({ ... options ... }) .on('success.form.fv', function(e) { // Prevent form submission e.preventDefault(); // Get the form instance var $form = $(e.target); // Get the FormValidation instance var bv = $form.data('formValidation'); // Use Ajax to submit form data $.post($form.attr('action'), $form.serialize(), function(result) { // ... Process the result ... }, 'json'); }); }); {% endhighlight %}

In the example above, you can see the form is prevented from doing default submission by using

{% highlight javascript %} e.preventDefault(); {% endhighlight %}
