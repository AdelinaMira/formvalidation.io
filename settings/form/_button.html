button

{% highlight javascript %} button: { // The submit buttons selector selector: '[type="submit"]', // The disabled class disabled: '' } {% endhighlight %}

selector — The submit buttons selector. These buttons will be disabled when the form input are invalid to prevent the valid form from multiple submissions.

disabled — The CSS class for disabled button. The default value is defined by specific frameworks as following:

Framework Disabled button class
Bootstrap disabled
Foundation disabled
Pure pure-button-disabled
Semantic disabled
UIKit disabled
