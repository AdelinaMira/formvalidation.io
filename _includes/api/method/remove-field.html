removeField

removeField(field*): BootstrapValidator — Remove given field

Parameter Type Description
field String|jQuery The field name or field element

By triggering the removed.field.bv event, you can perform additional task after removing given field:

{% highlight javascript %} $(document).ready(function() { $(form) .bootstrapValidator(options) .on('removed.field.bv', function(e, data) { // $(e.target) --> The form instance // $(e.target).data('bootstrapValidator') // --> The BootstrapValidator instance // data.field --> The field name // data.element --> The new field element // Do something ... }); }); {% endhighlight %}
