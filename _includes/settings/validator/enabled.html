enabled: Boolean

Indicate the validator is enabled or disabled. It is true, by default.

Look at the field's enabled option if you want to enable/disable all validators.

The table below shows three equivalent ways to enable or disable given validator:

Usage Example
HTML 5 attribute {% highlight html %} {% endhighlight %}
Plugin option {% highlight javascript %} $(document).ready(function() { $(form).bootstrapValidator({ fields: { fullName: { validators: { notEmpty: { enabled: true // or false } } } } }); }); {% endhighlight %}
enableFieldValidators() {% highlight javascript %} // Enable validator $(form) .bootstrapValidator('enableFieldValidators', fullName, true, 'notEmpty'); // Disable validator $(form) .bootstrapValidator('enableFieldValidators', fullName, false, 'notEmpty'); {% endhighlight %}
