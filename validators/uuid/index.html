Options

Option HTML attribute Type Description
message data-bv-uuid-message String The error message
version data-bv-uuid-version String The UUID version. Can be 3, 4, 5 or all default
When setting options via HTML attributes, remember to enable the validator by setting data-bv-uuid="true".
